## All Tasks Which Can Be Completed

Tasks Seller can complete:

- Login or register
- Create a bundle
- View forecast of bundle &rarr; predicted reservations and expected no-show probability, as well as recommendations
- Post bundle
- View all bundles, past and present
- Edit posted bundles
- Filter to find bundles &rarr; date, category or outcome status (Including “All” for specific filters)
- View all bundle reservations
- Update preparation status of reserved bundles &rarr; “Ready-to-collect”
- Update outcome status of reserved bundles &rarr; “Collected”, “No-show”, “Expired”
- See reports associated with bundles
- Respond to reports
- Highlight reports to Developers?
- Be notified of new reports leading back to bundle it belongs to in Reports
- Find bundles from collection code through search
- Validate collections through collection code
- Filter through reserved bundles by date, category, outcome status or preparation status
- Make report to developers
- View forecast analytics
- Edit company details

Tasks a Consumer can complete:

- Login/register
- Scroll main page of bundles
- Filter bundles &rarr; category, time-slot, “without certain allergy” or location
- Click on bundle and see more information &rarr; Content description
- Go to checkout
- Add report to bundle (open pop-up) &rarr; for collection preferences or notes (accessibility)
- Reserve bundle
- Receive collection code (pop-up with reservation details)
- Get notification on profile directing to reserved bundles list
- View reserved bundle list
- Filter through past and present reserved bundles, defaulted on today and uncollected bundles
- View collection code on reserved uncollected bundle
- Make report to bundle from list
- Get notification on profile if streak is updated
- View streak on profile
- Streak highlighted if streak updated
- See number representation of streak
- Be notified of bundle preparation status (”Reservation-Received”, “Being-Prepared”, “Ready-to-collection”)
- Receive badges
- View badges on profile
- Be notified of new badges on profile
- See impact report
- See small stats from report on profile (e.g. 15 meals saved so far, CO2 saved)
- See more detailed impact report
- See preparation status through reserved bundle list
- Change accessibility settings (dark mode, dyslexia friendly font, aria labels, the ability to navigate it via a keyboard, an easily readable contrast between elements
- Edit personal details

Tasks a Developer can complete (Incomplete):

- Login
- See all Sellers
- See all Seller reports
- Enter Seller profile
- Developer view of profile
- See Seller View of profile
- See Consumer reports of Seller
- Reply to reports from Seller
- Reply to reports from Consumer of Seller?

## Sprint 1 --

seller posts bundle &rarr; consumer reserves &rarr; claim code
generated &rarr; seller marks collected &rarr; consumer streak updates.

Seller:

- Logs-in or register
- On Seller's dashboard
- Clicks create bundle
- Creates bundle (Fill in essential fields, category, pick-up time)
- Finalize Bundle - Click complete, see summary of info
- Click Edit to correct anything
- Click Post to post the bundle
- Is back to empty bundle post
- Clicks on Bundle Posts
- See List of bundle posts &rarr; Buttons/options on bundles: Edit, Delete, Input code,
- Able to filter bundle options: Day (Today, yesterday, 1 week, 1 month), prep status, outcome status
- Clicks on Reserved Bundles
- See Bundle Reservations &rarr; Buttons/options on reservations: Prep status (default on "Unprepared"), outcome status, verify code (Give confirmation and automatically marks as collected), Link to reports on bundle
- Click Reports on bundle
- View bundle reports &rarr; clicks back and is returned to original place in posted bundles
- Filter reserved bundles &rarr; Outcome status (default on "Unprocessed"), Prep status (default on "Unprepared")
- Search reserved bundles by collection code (Search bar at top or side)
- Clicks Reports &rarr; Sees all reports to all bundles in compact list

Consumer

- Logs-in or register
- On main page
- Scrolls the main page or searches the main page with filters
- See “Filter” bar on side or at top &rarr; choose filters like price, company, location
- Clicks on “See more” for desired bundle - See more info (See less to see less)
- Clicks on "Reserve"
- Taken to checkout, able to click link to add report (e.g. need accessible pick-up) &rarr; when complete taken back to checkout
- Clicks "Confirm" to complete transaction
- Receives pop-up with bundle details and pick-up code and taken back to main page
- Notification on user profile indicating where the bundle information can be re-accessed
- Clicks user profile
- Clicks “Reserved Bundles” to see pending bundles &rarr; can see code on preview of bundle.
- Click link on bundle to make report
- Click link on profile to make separate report?
- Click on "Impact Report" to see more detailed impact report data
